# Calendário 2026 — Feriados do Brasil

Calendário visual de 2026 com todos os **feriados nacionais** e **pontos facultativos** oficiais para a administração pública federal, conforme a [Portaria MGI nº 11.460/2025](https://www.gov.br/gestao/pt-br/assuntos/noticias/2025/dezembro/confira-o-calendario-oficial-de-feriados-nacionais-e-pontos-facultativos-em-2026).

## Como usar

Abra o arquivo `index.html` no navegador (duplo clique ou arraste para o Chrome/Edge/Firefox).

**Pontos facultativos:** Carnaval (16–18/fev), 20/abr, Corpus Christi (04/jun), 05/jun, Dia do Servidor (28/out), vésperas de Natal e Ano Novo (24 e 31/dez).

> Feriados estaduais e municipais não estão incluídos. Consulte a legislação da sua cidade/estado para datas adicionais.
